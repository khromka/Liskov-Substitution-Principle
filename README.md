# Liskov-Substitution-Principle
«Об’єкти підкласів повинні бути взаємозамінними з об’єктами їх базових класів».  Тобто, якщо клас B успадковує клас A, то ми повинні мати можливість використовувати B замість A без порушення коректності програми. Це забезпечує узгодженість поведінки та правильний поліморфізм.
